# 📋 Sistema de Ordem de Serviço

Sistema desenvolvido em Java para gerenciamento de ordens de serviço de nobreaks e estabilizadores.

## 🚀 Funcionalidades

- Cadastro de ordens de serviço
- Checklist técnico
- Armazenamento em arquivo
- Listagem de ordens

## 🛠️ Tecnologias

- Java
- Programação Orientada a Objetos
- Manipulação de arquivos

## 📂 Estrutura

- model → entidades
- service → regras de negócio
- repository → persistência
- util → auxiliares

## 💡 Futuras melhorias

- API REST com Spring Boot
- Banco de dados
- Interface gráfica
